sasoptpy.Variable.get_value¶

Variable.get_value()¶

Returns the value of the expression after variable values are changed

Returns:

float

Value of the expression

Examples

>>> profit = so.Expression(5 * sales - 3 * material)
>>> m.solve()
>>> print(profit.get_value())
41.0
