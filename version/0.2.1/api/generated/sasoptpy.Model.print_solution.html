sasoptpy.Model.print_solution¶

Model.print_solution()[source]¶

Prints the current values of the variables

See also

Model.get_solution()

Notes

  • This function may not work for abstract variables and nonlinear models.

Examples

>>> m.solve()
>>> m.print_solution()
x: 2.0
y: 0.0
