sasoptpy.Constraint.get_value¶

Constraint.get_value(rhs=False)[source]¶

Returns the current value of the constraint

Parameters:
rhs : boolean, optional

Whether constant values (RHS) will be included in the value or not. Default is false

Examples

>>> x = so.Variable(name='x', init=2)
>>> c = so.Constraint(x ** 2 + 2 * x <= 15, name='c')
>>> print(c.get_value())
8
>>> print(c.get_value(rhs=True))
-7
