sasoptpy.reset_globals¶

sasoptpy.reset_globals()[source]¶

Deletes the references inside the global dictionary and restarts counters

See also

get_namespace()

Examples

>>> import sasoptpy as so
>>> m = so.Model(name='my_model')
>>> print(so.get_namespace())
Global namespace:
    Model
           0 my_model <class 'sasoptpy.model.Model'>,               sasoptpy.Model(name='my_model', session=None)
    VariableGroup
    ConstraintGroup
    Expression
    Variable
    Constraint
>>> so.reset_globals()
>>> print(so.get_namespace())
Global namespace:
    Model
    VariableGroup
    ConstraintGroup
    Expression
    Variable
    Constraint
